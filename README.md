# Stock [DON'T BUY THE STOCKS!]

---

## What is it?
Stock is a cli tool for watching stocks.

## How to use it?

### install
```bash
$ go install github.com/lemonyxk/stock
```

### run
```bash
$ stock -c [stock_code: 000001] -a [area: sh]
$ stock -c [stock_code: AAPL] -a [area: us]
```
